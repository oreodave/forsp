An implementation of the Forsp language.
Goals:
- Fully functional intepreter, with REPL and file reading
- A compiler implementation, targeting x86-64
- Extensions to standard language.  To be fleshed out further, but
  some ideas are:
  - A module system to link up external code files
  - FFI
  - Records
